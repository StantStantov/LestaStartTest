# TF-IDF

## Stack
- Go, net/http
- Postgresql
- Justfile
- Docker
- Swagger

## Getting started
1. Clone repo
```
git clone https://github.com/StantStantov/LestaStartTest.git
cd LestaStartTest
```
2. Use Justfile to build project
```
just build-test
just build-dev
just build-prod
```

## Swagger
API will be presen on `/swagger/index.html` after start.
